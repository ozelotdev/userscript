## musasi.jp

### キーボードショートカット

Download: [keyboard-shortcut.user.js](https://github.com/ozelotdev/userscript/raw/main/musasi.jp/keyboard-shortcut.user.js)

| キー  | 機能                  |
| ----- | --------------------- |
| 1     | ⭕️ を選択             |
| 2     | ❌ を選択             |
| 3     | 「まよった」 を選択   |
| Enter | 「次の問題へ」 を選択 |
